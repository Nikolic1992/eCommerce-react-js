# 🛍️ eCommerce Website  
Simple and modern eCommerce website built with React and Vite.  
It uses Tailwind CSS and Material UI for styling, and includes routing, icons, and toast notifications.

## 🧰 Tech Used  
⚛️ React + Vite  
🎨 Tailwind CSS  
🧱 Material UI  
🧭 React Router DOM  
🌟 React Icons  
🔔 React Toastify  

## ✨ Main Features  
🛍️ Product listing  
➕ Add to cart  
🔍 Product search  
🔄 Page routing  
🔔 Toast messages  
📱 Responsive design  
🔐 User login & signup with Clerk
